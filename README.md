# การเบิกถอนเงิน 

หัวข้อ: ระบบ “การเบิกถอนเงิน & รายงานสถานะการถอน”

กำหนดส่ง: ภายในวันพุธ เวลา 18:00
ส่งเป็น: GitHub repo หรือ zip file พร้อม README รายละเอียดตามไฟล์แนบ

โจทย์สรุป:
- ทำ Single Page App (React/Next.js, แนะนำ TypeScript) ที่มี:
  1) หน้ารายการคำขอเบิกถอน (list) พร้อมสถานะ
  2) ฟอร์มส่งคำขอเบิกถอนใหม่ (support upload รูป/วิดีโอ)
  3) หน้ารายละเอียดแต่ละรายการ (แสดง history + attachments)
  4) filter / search by status/name/tx id



** วิธีรัน ** 
1. ติดตั้ง npm install
2. npm run dev
3. เปิดเบราว์เซอร์ http://localhost:3000



** front-end การเบิกถอนเงิน ** (React + Vite + TypeScript)

ไฟล์ App.tsx เป็นไฟล์การจัดการข้อมูลต่างๆ เช่น การสร้างฟอร์มคำขอ , รายการ , การค้นหา 
ไฟล์ App.css เป็นสไตล์ css ทั้งหมด
ไฟล์ mockApi.ts เป็นไฟล์ mock API 
โฟลเดอร์ public เป็นรูปไอคอนและรูปภาพที่อัปโหลดไว้เพื่อมาใช้ใน Mock Data 

ในหน้า UI 
- แสดงช่อง Input การค้นหาชื่อ/TX ID/รายการ 
- แสดง filter สถานะ ได้แก่ รออนุมัติ/อนุมัติ/ปฏิเสธ
- ตารางรายการเบิกถอน แสดงข้อมูล TX ID/ชื่อ/รายละเอียด/จำนวน/สถานะ/วันที่ คลิกเข้าไปที่รายการเบิกถอนที่ต้องการเพื่อดูรายละเอียดเพิ่มเติม TX ID/ชื่อ/รายละเอียด/จำนวน/สถานะ/วันที่/รูปภาพหรือวิดีโอที่อัปโหลด 
- ฟอร์มคำขอเบิกถอนเงินที่ + แสดง Popup ให้กรอกข้อมูล ชื่อผู้ขอ/รายละเอียดเหตุผลการเบิกถอน/จำนวน (บาท) และอัปโหลดรูปภาพหรือวิดีโอรองรับหลายไฟล์
